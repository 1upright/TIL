## 조건

- if, elif, else
- while문

```python
if True:
    print('조건문입니다.')
```

```python
if dust > 50:
    print('50초과')
else:
    print('50이하')
```

```python
greeting = 'hi'
i = 0
while i < 10:
	print(greeting)
	i += 1
```

```python
greeting = 'hi'

for i in range(10):
    print(greeting)
```
